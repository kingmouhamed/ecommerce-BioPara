# Ecommerce BioPara

Projet Front-End d’un site e-commerce de produits de parapharmacie.

## Objectif
Afficher les produits avec filtrage par catégorie via l’URL.

## Fonctionnalités
- Liste des produits
- Filtrage par catégorie (?category=Parapharmacie)
- Interface responsive

## Technologies
- React
- Vite
- JavaScript
- CSS

## Lancer le projet
npm install
npm run dev
